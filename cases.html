Прототип AI · многоагентная архитектура

AI-генератор коммерческих предложений

Многоагентный генератор КП из техзаданий. На входе — ТЗ заказчика (PDF/DOCX), на выходе — готовое коммерческое предложение по шаблонам исполнителя с расчётом по актуальной базе материалов и логистики. Локальный деплой, фиксированная стоимость пилота.

Архитектура: парсер ТЗ + композер КП + верификатор. Все цифры проверяются по базе исполнителя. Работает с русским языком, форматами PDF/Excel/Word.

Эффект
Подготовка КП
с 5–7 дней до 1–2 дней
Действующий инструмент Отгружено несколько документов

Генератор технико-коммерческих предложений

Действующий инструмент-шаблон для генерации ТКП в отрасли пожарной безопасности. Шаблоны под СПЗ, СОУЭ, АПТ. AI-заполнение по входным данным заказчика, расчёты, юридические формулировки.

Переиспользуемый инструмент — после первого внедрения каждый следующий ТКП готовится в разы быстрее. Адаптируется под другие отрасли.

Эффект
Снижение трудозатрат
на подготовку ТКП в 3–5 раз
Активная разработка RAG · bge-m3 + reranker

AI-поиск по технической документации

Боевой пайплайн внутри диагностической SaaS-платформы. Embeddings bge-m3, reranker, целевые метрики Recall@5 ≥ 0.85. Переупаковывается под корпоративный поиск с минимальными доработками.

Работает с регламентами, инструкциями, паспортами оборудования. Естественный язык на входе — релевантный ответ со ссылкой на исходный документ.

Эффект

с часов до секунд
